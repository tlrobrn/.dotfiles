# .dotfiles

personal dotfiles for mac and pco cloud-box

## Requirements

(zsh and ruby)

### This Repo

```sh
git clone git@github.com:tlrobrn/.dotfiles.git
```

### RCM

Mac:
```sh
brew install rcm
```

Cloud-Box:
```sh
sudo apt-get -y install rcm
```

## Usage

Mac:
```sh
RCRC=$HOME/.dotfiles/rcrc rcup -t mac
```

Cloud-Box:
```sh
RCRC=$HOME/.dotfiles/rcrc rcup -t pco
```
